# FootDrafts

FootDrafts is a Ruby on Rails web application boilerplate with a React frontend.

## Stack

- Ruby on Rails 7
- React 19
- esbuild (JavaScript bundling)
- Tailwind CSS
- SQLite (default development/test database)

## Getting started

```bash
bundle install
yarn install
bin/setup
bin/dev
```

Then open `http://localhost:3000`.

## Running tests

```bash
bundle exec rails test
```
